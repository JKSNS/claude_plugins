# Skill Builder

Create custom `/slash` commands from natural language descriptions. Describe what you want the command to do and it generates the SKILL.md file, installs it, and makes it available immediately.

## Install

```
/install-plugin skill-builder
```

## Usage

Tell Claude what you want the command to do in plain language. It creates the SKILL.md with frontmatter, usage docs, and step-by-step execution instructions, then installs it to `~/.claude/skills/`.

See [skills/](../../skills/) for how to build them manually.
